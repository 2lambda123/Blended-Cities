Detailed Description

the main city elements collection

store any element of the city, including outlines. allows 'real' object / element lookups

Parameters:
nameused internally as key for parenting and element lookup ! shouldn't be modified !
typeused for lookup between the element as element and the same element as builder or outline. I don't think it's useful anymore, since the name gives the info about the class of the element.
pointeris the pointer to the real object. set to -1 if the object is missing. i fact it correspond to the obj.as_pointer() function. it allows to change the name of the real object without breaking the relation-ship between the element and the object.
